Writing Dependencies

Ah yes, the best part about this whole thing; writing your own functions to be included as dependencies through the Dependency Factory. It couldn't be easier, all you need to do is put your function(s) name(s) into the addDeps() function. This function has two arguments; the first is to define whether it needs run before or after pageload, the second to hold the first function, but you can pass as many parameters as you want after that for function names and it will run and add them.

NEAT FEATURE ALERT! Did you know when your function is called by the dependency_factory it will automatically assign your functions context to the window? Yup, you heard right, you have access to all the window functions provided by the web browser, so you can interact with the page itself. If you prefer to change the context just assign a new variable the context you want to use, preferably named that so there is separation between this context and that context. Onto the next section
