Setup

With every webpage body must come a hidden header that links all the files required to run the page, and that's no different here. The tool relies on all prexisting webpage design it just refactors how the custom elements will show up, the DOM can stil recognize any tag with text and attributes as an element even if it isn't predefined as a web element. HTML is just a special kind of XML data structure that supports comments so it makes sense to be able to do things like this. To use PLACSS, you need to first get your HTML frame started with the usual <!DOCTYPE html>, <head>, and <body> tags. There is a copiable template located on the navbar if you would like to skip the nitty gritty.

PLACSS uses a Plaster JS library import called dependency_factory.js. This is a special library where you can even add your own dependencies but we will cover that later. So first include the dependency factory as one of the first scripts in your document via <script src="PATH_TO_/dependency_factory.js"></script>. Even before all of this you should link in the CSS via <link rel="stylesheet" type="text/css" media="screen" href="PATH_TO_/placss.css" /> so the tags can get their predefined css properties. Now after the dependency_factory.js insert <script src="PATH/placss.js></script>. The final thing you will need is to have another script tag run the installDeps(); function to install the added dependencies. You are now free to use PLACSS, we will cover syntax and features in the next section.
